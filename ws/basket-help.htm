Справка по экрану "Заказ"

Оформление Заказа производится на трех, связанных переходами, экранах:

  • Заказ
  • Выбор товаров
  • Характеристики товара

На этом экране нужно указать параметры "шапки" Заказа:

  • Дата - дата предполагаемой отгрузки Заказа;
  • Клиент - выбор клиента, которому будет отгружен Заказ;
  • Торговая Точка - выбор торговой точки клиента, в которую будет отгружен Заказ. Если выбранный клиент не имеет торговых точек, это поле скрывается;
  • * Предприятие - выбор юридического лица, на которого будет оформлен Заказ;
  • * Тип Оплаты - выбор типа оплаты для Заказа;
  • Примечания - любые примечания к Заказу;
  • Информация о том, сколько выбрано товаров в Заказ и на какую сумму
* параметры определяются конфигурацией программы и могут отсутствовать

При нажатии на кнопку "Сохранить" производится проверка правильности оформления Заказа, и, если все сделано правильно, Заказ сохраняется в Списке Заказов и производится переход на экран, из которого пришли сюда.

Если проверка завершилась с ошибкой, внизу всплывает сообщение об ошибке. Список ошибочных ситуаций:

  • Не выбраны товары в Заказе;
  • Не установлена дата в Заказе;
  • Не выбран клиент в Заказе;

Кнопка "Сохранить" может отсутствовать, если Заказ корректируется в состоянии "Только для чтения"

В первой строке на экране слева направо присутствуют:

  • Кнопка "назад" - переход к предыдущему экрану;
  • Название экрана;
  • Кнопка "I" - вызывает этот экран справки;
  • Кнопка "Товары" - вызывает экран Выбор товаров

Справка по экрану "Выбор товаров"

Выбор товаров - это второй из трех экранов для формирования Заказа:

  • Заказ
  • Выбор товаров
  • Характеристики товара

Этот экран предназначен для выбора товаров в Заказ.

Экран существует в двух видах: "Товар" и "Корзина".
В первой строке на экране "Товар" слева направо присутствуют:

  • Кнопка "назад" - переход к предыдущему экрану;
  • Название экрана;
  • Кнопка "I" - вызывает этот экран справки;
  • Кнопка "Корзина" - меняет вид окна на содержимое Корзины, т.е. вызывает экран "Корзина"

При нажатии на кнопку "Товары", расположенную в правом верхнем углу экрана, видим окно для поиска (фильтрации) товара по категориям. Нажатие на стрелку "вправо" в правом углу вызывает список категорий товаров, по которым можно сделать выбор категории. Кроме того, в строке поиска (текстовый блок "Поиск" в левом углу экрана) можно сделать дополнительную фильтрацию товара в уже выбранной категории. После такой фильтрации видим список товаров, из которых можно сформировать Заказ.
При нажатии на конкретный товар мы попадаем на следующий экран - Характеристики товара

В первой строке на экране "Корзина" слева направо присутствуют:

  • Кнопка "назад" - переход к предыдущему экрану;
  • Название экрана;
  • Кнопка "I" - вызывает этот экран справки;
  • Кнопка "Товар" - вызывает экран "Товар" с сохраненной фильтрацией товаров по категории.

Содержимое Корзины - это список товаров с указанием их количества, уже набранных в конкретный Заказ. Первая строка списка содержит информацию об общем количестве выбранного товара и общей сумме Заказа.
